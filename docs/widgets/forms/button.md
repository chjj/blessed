#### Button (from Input)

A button which can be focused and allows key and mouse input.

##### Options:

- Inherits all from Input.

##### Properties:

- Inherits all from Input.

##### Events:

- Inherits all from Input.
- __press__ - Received when the button is clicked/pressed.

##### Methods:

- Inherits all from Input.
- __press()__ - Press button. Emits `press`.
